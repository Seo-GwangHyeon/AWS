0124 AWS EC2 설정
==================
- 인스턴스 : 쉽게 생각하면 가상의 컴퓨터 1대를 말한다.
- SSH (Secure Shell Protocol): 유닉스 계열의 운영체제를 원격으로 사용하려면 ssh를 허용 해둬야한다.

0125 AWS EC2 원격 접속
============================
- Xshell을 이용해서 EC2 인스턴스에 접속하였다.
- Xshell은 개인 및 학교에서만 무료로 사용할 수 있다.
